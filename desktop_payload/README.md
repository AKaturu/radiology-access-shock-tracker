# Desktop Payload

This folder contains the compact reviewed analysis package bundled into desktop builds.

Current payload:

- Reviewed NC MQSA comparison: `2026-06-19` to `2026-06-20`
- Route package: self-hosted OSRM, driving profile
- Readiness audit: `READY`, 0 blockers, 0 warnings
- Result boundary: no-observed-change validation run, not a longitudinal trend claim

Users do not need Census or OpenRouteService API keys to open this bundled dashboard. Keys are only
needed when regenerating fresh Census inputs or hosted routing drafts.
